# Memory Bank Index

## Purpose
This memory bank serves as the project's long-term knowledge repository, tracking decisions, assumptions, key information, and project evolution for the Early Career Financial Strategy guide.

## Structure

### Core Documents
- **[PROJECT-CONTEXT.md](./PROJECT-CONTEXT.md)** - High-level project overview, goals, and target audience
- **[DECISIONS-LOG.md](./DECISIONS-LOG.md)** - All major decisions with rationale and date
- **[FINANCIAL-ASSUMPTIONS.md](./FINANCIAL-ASSUMPTIONS.md)** - Key financial parameters, rates, and calculations
- **[RESOURCES.md](./RESOURCES.md)** - External resources, tools, and references used
- **[ACTIVE-TASKS.md](./ACTIVE-TASKS.md)** - Current work items, TODOs, and future enhancements

### How to Use This Memory Bank

1. **Before starting work:** Review relevant memory bank files for context
2. **During work:** Update decision logs and assumptions as changes occur
3. **After completing tasks:** Document what was done and why in appropriate files
4. **For new features:** Check FINANCIAL-ASSUMPTIONS.md for consistent parameters

### Memory Bank Maintenance

- Update relevant files whenever significant decisions are made
- Keep ACTIVE-TASKS.md current with work in progress
- Archive completed major milestones in DECISIONS-LOG.md
- Review and refresh assumptions quarterly
